# Entrenamiento en Git

> Asignatura: Sistemas de Big Data  
> Curso de Especialización en Inteligencia Artificial y Big Data

## 📌 Descripción
Esta sección funciona como un bloque instrumental centrado en el uso de Git como estándar de la industria para el control de código en proyectos complejos. Abarca todos los fundamentos de configuración, ramificación y colaboración en directorios versionados antes de afrontar arquitecturas de Big Data.

## 🎯 Objetivos de Aprendizaje
- Comprender la filosofía de instantáneas del control de versiones distribuido.
- Implementar comandos esenciales de repositorio local y preparación de commits.
- Colaborar utilizando flujos de trabajo sobre repositorios remotos.

## 🧠 Contenidos
Exploración de la anatomía de un commit y la transición entre el directorio de trabajo, el área de staging (preparación) y el repositorio histórico. Se examina el manejo de la bifurcación (branches) para desarrollo en paralelo.
